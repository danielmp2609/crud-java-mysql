# CRUD de Usuarios - Java + MySQL

Aplicación de escritorio desarrollada en Java que permite la gestión de usuarios mediante operaciones CRUD, utilizando una base de datos MySQL y arquitectura por capas (DAO + Service).

## Tecnologías utilizadas

- Java
- MySQL
- JDBC
- Swing (JFrame)
- FlatLaf (UI moderna)

## Funcionalidades

- Crear usuarios
- Listar usuarios
- Actualizar usuarios
- Eliminar usuarios
- Buscar por nombre
- Validación de datos

## Arquitectura

El proyecto está estructurado en:
- model
- dao
- service
- view

## Cómo ejecutar

1. Clonar el repositorio
2. Configurar la base de datos
3. Ejecutar Main.java
